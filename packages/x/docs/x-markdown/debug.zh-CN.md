实时监控 Markdown 渲染过程中的关键性能指标（如 FPS 与内存占用），以浮层形式在页面上展示，辅助开发者识别渲染瓶颈。

## 代码演示

<code src="./demo/streaming/debug.tsx">性能监控面板</code>

## API

| 属性  | 说明                 | 类型      | 默认值  |
| ----- | -------------------- | --------- | ------- |
| debug | 是否启用性能监控面板 | `boolean` | `false` |

> ⚠️ 仅限开发环境使用，生产构建中请确保关闭以避免性能开销与信息泄露。
